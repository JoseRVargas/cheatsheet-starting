1. HTML

Tables

Definition

The HTML table model allows authors to arrange data - text, preformatted text, images, links, forms, form fields, other tables, etc. - into rows and columns of cells.

Examples

There are several tags that are used to create tables in HTML. The main ones are listed below.

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows containing actual table data.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Entities

Definition

HTML entities are special characters that are reserved for use in an HTML document. They are used to render reserved characters in HTML.

Examples

There are several entities that are used to render reserved characters in HTML. The main ones are listed below.

> Greater than (end of HTML element) &gt;
& Ampersand &amp;

' Single quote &apos;

© Copyright &copy;
® Registered &reg;
™ Trademark &trade;
× Times &times;
÷ Divide &divide;

2. CSS

Selector

Definition

CSS selectors are used to select the content you want to style. Selectors are the part of CSS rule set that actually selects the content you want to style.

Examples

There are several selectors that are used to select the content you want to style in CSS. The main ones are listed below.

Selector Description
* {} Selects all elements
h1 {} Selects all <h1> elements
#id {} Selects the element with id="id"
.class {} Selects all elements with class="class"
element, element {} Selects all element elements and all element elements
element element {} Selects all element elements inside element elements
element > element {} Selects all element elements where the parent is a element element
element + element {} Selects all element elements that are placed immediately after element elements
element ~ element {} Selects every element element that is preceded by a element element
:link {} Selects all unvisited links
:visited {} Selects all visited links
:active {} Selects the active link
:hover {} Selects links on mouse over
:focus {} Selects the input element that has focus
:first-child {} Selects every element element that is the first child of its parent
:nth-child(n) {} Selects every element element that is the nth child of its parent
:nth-last-child(n) {} Selects every element element that is the nth child of its parent, counting from the last child
:first-of-type {} Selects every element element that is the first element element of its parent

Properties

Definition

CSS properties are used to specify the style of the HTML elements. They are used to set the values of the style sheet. CSS properties are used to set the look of the HTML elements.

Examples

There are several properties that are used to specify the style of the HTML elements in CSS. The main ones are listed below.

Property Description
color Specifies the text color of an element
font-family Specifies the font of an element
font-size Specifies the size of the font
font-style Specifies the style of the font
font-weight Specifies the weight of the font
text-align Specifies the horizontal alignment of text
text-decoration Specifies the decoration added to text
text-transform Specifies the capitalization of text
letter-spacing Specifies the space between characters in a text
line-height Specifies the line height
background-color Specifies the background color of an element
background-image Specifies the background image of an element
background-repeat Specifies how the background image should be repeated
background-attachment Specifies whether the background image should be fixed or scroll with the rest of the page
background-position Specifies the position of the background image
border A shorthand property for setting all the border properties in one declaration
border-color Specifies the color of the border
border-style Specifies the style of the border
border-width Specifies the width of the border
border-radius A shorthand property for setting all the four border-*-radius properties
padding A shorthand property for setting all the padding properties in one declaration
margin A shorthand property for setting all the margin properties in one declaration
width Specifies the width of an element
height Specifies the height of an element
display Specifies the type of rendering box used for an element
visibility Specifies whether or not an element is visible
overflow Specifies what happens if content overflows an element's box
position Specifies the type of positioning method used for an element
z-index Specifies the stack order of an element
float Specifies whether or not a box should float
clear Specifies what elements can float beside the cleared element and on which side
left Specifies the left position of a positioned element
right Specifies the right position of a positioned element
top Specifies the top position of a positioned element
bottom Specifies the bottom position of a positioned element
animation A shorthand property for setting all the animation properties in one declaration
transition A shorthand property for setting all the transition properties in one declaration
transform Applies a 2D or 3D transformation to an element
box-shadow Attaches one or more shadows to an element
filter Defines effects (e.g. blurring or color shifting) on an element before the element is displayed

3. JavaScript

Functions

Definition

A JavaScript function is a block of code designed to perform a particular task. It is executed when "something" invokes it (calls it).

Examples

There are several functions that are used to perform a particular task in JavaScript. The main ones are listed below.

Function Description
alert() Displays an alert box with a message and an OK button
prompt() Displays a dialog box that prompts the visitor for input
confirm() Displays a dialog box with a specified message, along with an OK and a Cancel button
console.log() Writes a message to the console
document.write() Writes HTML expressions or JavaScript code to a document
open() Opens a new browser window
close() Closes the current browser window
write() and writeln() Writes HTML expressions or JavaScript code to a document
getElementById() Returns the element that has the ID attribute with the specified value
getElementsByTagName() Returns a collection of all elements in the document with the specified tag name
getElementsByClassName() Returns a collection of all elements in the document with the specified class name
removeChild() Removes a child node from an element
replaceChild() Replaces a child node in an element
appendChild() Adds a new child node to an element
addEventListener() Attaches an event handler to the specified element
removeEventListener() Removes an event handler that has been attached with the addEventListener() method
